# Drafts
Following GitHub's  instruction  *About READMEs*, the reader will find here the answers to some questions about this repo.

#### 1) What does the project do?
This project aims to contain different drafts of both markdown files, created for practice using this editing mode, and C-language where i solve some exercices for the same reason.

For the Markdown section, it should include at least one "MainTips" file with a summary of the basic commands one should now to start writing in markdown.

#### 2) Why is the project useful?
For beginners it can be useful to see how to learn markdown in a simple way and (mainly) for the author it is useful for starting the use of GitHub and markdown editors.

#### 3) How can users start working on the project?
The project is intended for the main purpose of the author's personal use, but if someone would like to contribute, their work will be  welcome.

#### 4) Where users can find assistance for this project? 
For the same reason as above, I think that none will need assistance, but in case they do, I could be reached through the email address provided in the README file available in the overview

#### 5) Who mantains and contrbutes to this project?
Anyone can contribute to this project.
